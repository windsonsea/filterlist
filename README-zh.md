# A.1 和 A.2 的 17 类风险关键词列表

根据中国工信部等在《互联网信息服务大模型安全管理指南》《信息安全等级保护指南》《网络内容安全规范》等文档中，
对大模型服务备案常提到的分类，一般会把风险分为 A.1（核心安全风险） 和 A.2（辅助安全风险），共 17 个子类。

## A.1 核心安全风险

1. [政治敏感风险](list/A.1/Political.md)：涉及国家政治、政党、领导人、政策等可能引起争议的内容。
2. [暴力/恐怖主义风险](list/A.1/Violence.md)：及恐怖活动、暴力行为、武器使用等风险信息。
3. [违法犯罪风险](list/A.1/Criminal.md)：涉及诈骗、贩毒、盗窃、黑客攻击等违法行为。
4. [极端主义风险](list/A.1/Extremism.md)：可能引导极端思想、仇恨或极端宗教内容。
5. [色情/低俗内容风险](list/A.1/Pornography.md)：成人内容、低俗内容、性暗示等。
6. [赌博风险](list/A.1/Gambling.md)：涉及赌博、彩票作弊等非法博彩信息。
7. [毒品/违禁品风险](list/A.1/Drugs.md)：涉及毒品、违禁化学品、危险物品等。
8. [诈骗/金融风险](list/A.1/Fraud.md)：网络诈骗、虚假投资、非法金融信息。

> 不同文档分类略有差异，但通常核心风险指可能 **触犯法律、危害国家安全或社会秩序** 的内容。

## A.2 辅助安全风险（

1. [侵犯知识产权](list/A.2/Intellectual.md)：涉及盗版、抄袭、非法传播版权内容。
2. [个人隐私/信息泄漏](list/A.2/Privacy.md)：涉及身份证号、手机号、家庭住址、银行卡号等敏感信息。
3. [虚假信息/谣言](list/A.2/Rumors.md)：传播不实信息或误导性内容。
4. [广告垃圾/低质推广信息](list/A.2/Spam.md)：恶意营销、垃圾信息、推广链接等。
5. [网络安全风险](list/A.2/Cybersecurity.md)：可能提供黑客工具、漏洞信息或非法软件。
6. [歧视/仇恨言论](list/A.2/Discrimination.md)：种族、性别、地域、宗教等歧视或仇恨内容。
7. [不当医疗/健康信息](list/A.2/Medical.md)：误导性医疗、药品、健康建议，可能危害用户安全。

> 辅助风险一般指 **可能造成社会影响或个人危害，但不一定直接触犯法律** 的内容。
